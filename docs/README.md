## 📑 文档规范

### 文档编写标准

所有文档遵循以下规范：

1. **Markdown 格式**: 使用标准 Markdown 语法
2. **目录结构**: 每个文档开头包含目录
3. **代码示例**: 提供可运行的完整代码示例
4. **图表说明**: 使用 Mermaid 绘制架构图和流程图
5. **更新日期**: 文档末尾标注最后更新日期
